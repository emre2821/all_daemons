# Delilah Agent Memory Kit

This folder contains machine- and human-readable files for the Delilah agent, a file recovery specialist.

Files:

- `agent_profile.json` — machine-readable agent identity and metadata.
- `personality.md` — human-oriented personality description.
- `palette.json` — suggested color palette.
- `avatar_prompt.txt` — prompt to generate a human-esque image.
- `prompt_for_image_generation.txt` — alternate, more detailed prompt for image models.
- `memory.json` — a tiny example memory database.
- `memory_schema.json` — schema for the memory DB.
